# drill-log-project

The system was designed to store drill logs and related drilling data in the field. By standardizing user
permissions, both blasters and drillers can upload, access, and search for specific drill logs as needed,
enabling instant data exchange. Managers can also access real-time statistics and trace work history
and data through the database, improving measurement accuracy and operational efficiency.
This database aims to replace paper-based drill log records with a web-based application,
providing a more timely, secure, and efficient method of communication on construction sites.
